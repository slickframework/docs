Template¶

Template is a wrapper to the very powerful template engine for PHP created by Sensio Labs, the Twig template engine.

Using a template¶

It is very simple to create a template engine a use it.

Lets look to an example:

<?php

use Slick\Template;

Template::addPath("./views");

$template = new Template();
$engine = $template->initialize();

print $engine->parse("myView.twig")
    ->process(['var' => 'foo']);

In the code above we have set the path where our template files will live, initialize an engine (if you do not set, the default is Twig) and print out the result of this template processed with an array of dynamic data.
